# Smart Home Automation System

This project implements a smart home and smart plant automation system using Arduino and simulated in Tinkercad.

## Features
- Gas detection with LED warning and buzzer alert
- Motion-based lighting control
- Temperature-based fan activation
- Soil moisture monitoring with automatic water pump

## Sensors Used
- PIR Motion Sensor
- Gas Sensor
- Soil Moisture Sensor
- Temperature Sensor
- LDR (Light Sensor)

## Components
- Arduino Uno
- Buzzer
- LEDs
- Fan
- Water Pump
